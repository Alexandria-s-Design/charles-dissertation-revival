# Charles Dissertation Revival

This repository packages Dr. Charles Martin's dissertation and the working materials needed to revive it into a current publication, thought leadership, and product/service platform.

## Purpose

The project focus is not interactive whiteboards as a device. The enduring value is the instructional framework behind the dissertation:

- vocabulary development for multilingual learners
- multimodal instruction
- pedagogy-first technology integration
- instructional design for language acquisition
- AI-era reinterpretation of earlier edtech research

## Contents

- `source/` - primary source files, including the dissertation PDF and archived analysis session
- `docs/contractor-brief.md` - project brief for contractor execution
- `docs/12-month-roadmap.md` - year-long roadmap
- `docs/working-brief.md` - concise strategic summary
- `docs/email-to-contractor.md` - send-ready email draft
- `docs/source-index.md` - links, repos, and research starting points

## Lead Framing

Primary article concept:

`From Interactive Whiteboards to AI Tutors: A 10-Year Retrospective on Technology-Enhanced Vocabulary Instruction for English Language Learners`

## Repo Intent

This repo is meant to be shareable with a contractor who does not have access to the local machine. It should contain enough context for research, planning, publication targeting, and execution support.
